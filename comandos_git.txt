git init . >> cria um repo git na pasta atual
git status >> mostra status
git add >> adicona arquivo para comitar
git commit >> faz commit do q estava pra er comitado
git log >> mostra historico commit

fluxo de trabalho
1 criar o dir do projeto
2 iniciar o git dentro do diretorio criado (git init)
3 editar ou criar arquivos
4 git status
5 git add
6 git status
7 git commit -m "mensagem"
8 git status
9 volta para o passo 3
